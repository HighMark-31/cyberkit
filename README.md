## ğŸš€ Getting Started

### Quick Demo

Try CyberKit online: **[cyberkit.mylaby.space](https://cyberkit.mylaby.space)**

### Local Installation

**Prerequisites:**
- Node.js 16+ (or Bun 1.0+)
- npm, yarn, or Bun package manager

**Step 1: Clone the repository**

```bash
git clone https://github.com/HighMark-31/cyberkit.git
cd cyberkit
```

**Step 2: Install dependencies**

```bash
# Using npm
npm install

# OR using yarn
yarn install

# OR using Bun (fastest)
bun install
```

**Step 3: Start development server**

```bash
# Using npm
npm run dev

# OR using yarn
yarn dev

# OR using Bun
bun run dev
```

The app will be available at `http://localhost:8080` (or similar port shown in terminal)

**Step 4: Build for production**

```bash
npm run build    # Creates optimized build in 'dist/'
```
