# Changelog

## v2.0

- **System & Terminal**: Added fully functional Linux Terminal simulator and Chmod Calculator.
- **OSINT Capabilities**: Added Sherlock Web (username search), Subdomain Finder, and Google Dorks generator.

- **Network Analysis**: Added Advanced DNS Recon and Subnet Calculator.
